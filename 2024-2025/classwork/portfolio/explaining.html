HTML fieldset

The fieldset tag in HTML is used to make a group of related elements in the form.

HTML legend

The Legend element is used in HTML forms to provide a caption or a title for the associated fieldset element.

HTML Input tag (type text, date, range)

The HTML input tag defines the field where the user can enter data. The type attribute determines what type of user input is taken.

HTML Attributes

HTML Attributes helps modify and control HTML Attributes

HTML checkboxes

The input type="checkbox" defines a checkbox. The checkbox is shown as a square box that is ticked (checked) when activated. Checkboxes are used to let a user select one or more options of a limited number of choices.

HTML radio buttons

The HTML input type="radio" defines a radio button, grouped by sharing the same name attribute.

HTML textarea

HTML textarea is a element that is also used often as a form to collect user inputs and comments or reviews.

HTML Buttons

HTML Buttons is a tag that is clickable and that the user can use, A button could allow the user to submit such as forms etc.

HTML label

The label element is used to associate a text label with a form input field. The label is used to tell users the value that should be entered in the associated input field.

HTML Forms

The Form element is a container for different types of input elements, such as: text fields, checkboxes ans etc.
